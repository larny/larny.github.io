第一章 地球的性质、结构及物质组成

发表于 2017-05-17

地球物理性质

大地水准面

是指由平均海平面所构成并延伸通过陆地的封闭曲面，为假想的重力等位面。地球的形状是指大地水准面所圈闭的形状。

大陆地形特征

山地，丘陵，平原，高原，盆地

海底地形特征

大陆边缘

大陆和大洋盆地之间的过渡地带。包括大陆架、大陆坡、大陆基、岛弧和海沟。

  1. 大西洋型大陆边缘（被动大陆边缘）
  2. 太平洋型大陆边缘（主动大陆边缘）

    大陆中脊

    大洋盆地

地磁场

由基本磁场、变化磁场、磁异常 三个部分组成

地磁极性倒转

地球磁极在地质时期中的交替现象

地温场

是指地球内部各层中温度的分布状态，是地球内部空间各点在某一瞬间的温度总和

地壳浅层温度分层

变温层、 常温层、增温层

地温梯度 (地热增温率)

指地球深度每向下加深 100m 所增加的温度。
地热梯度的方向一般指向温度增加的方向，称正梯度；
如果温度随深度的增加反而降低，称负梯度。

地热流 (大地热流密度)

指单位时间单位面积内通过地球表面所散失的热量

地热异常

指地表热流值或地温梯度明显高于平均值或背景值的现象，这样的地区称为地热异常区。

地球内部圈层

波速不连续界面

当遇到不同介质的突变界面时，地震波射线就会发生反射和折射，而且其传播速度也会发生变化，这些界面称为波速不连续界面。
有莫霍洛维奇不连续面、古登堡不连续面

地壳

莫霍面以上的地球表层，主要由碳酸盐类岩石组成，是一个刚性的外壳。
分为 大陆地壳、大洋地壳

大陆地壳

双层结构，上部硅铝层（花岗质层）；下层硅镁层（玄武岩层）。两层之间的界面叫康拉德面

大洋地壳

单层结构，硅镁层组成，地层较薄，较年轻。

地壳重力均衡

指地壳物质为了适应重力的作用，总是力求与其更深部的物质之间达到质量或重量上的平衡状态的现象。重力异常越高，地壳越薄，莫霍界面越浅。

地幔

位于莫霍面以下、古登堡面以上的地球中间一个圈层，是地球的主体部分。

地核

位于古登堡面至地心的部分

地球外部圈层

大气圈

是因地球引力而聚集在地标周围的气体圈层。

大气圈自地面向上依次分为对流层、平流层、中间层、暖层、散逸层。

水圈

是地球表层的水体所构成的圈层，对地球表层环境的形成和改造起到重要作用。水分为海水、大气水、陆地水。

海水

  1. 波浪：最基本的运动形式，由风的吹刮形成，风与海面之间产生摩擦力，使海水产生运动形成波浪。
  2. 潮汐： 全球性海水周期性涨落现象。
  3. 海流： 沿一定方向有规律移动的海水。
  4. 浊流： 载有大量悬浮物质的高密度水下重力流

陆地水

  1. 地面流水

    水质点运动状态：层流、紊流、环流、涡流

  2. 地下水

    分为包带水、潜水、承压水

  3. 湖泊与沼泽水
  4. 冰川水

生物圈

是指地球表层由生物及其生命活动的地带所构成的连续圈层，是地球上所有生物以及其生存环境的总称。

  1. 原核生物界
  2. 原生生物界
  3. 真菌界
  4. 植物界
  5. 动物界

分类单位： 界、门、纲、目、科、属、种

地球的物质组成

克拉克值

某种元素在地壳中的平均百分含量

矿物

天然地质作用形成的固体化合物或单质，具有一定的化学成分和物理性质，是组成地球的基本物质。

晶体

天然产生的具有几何多面体形态、内部质点在三维空间做有规律的重复排列的固体

晶体格架

指晶体中原子、离子等重复排列的几何图形

晶体特性

自限性、均一性、对称性、稳定性、定熔性

同质多象和类质同象

同质多象

化学成分相同，但质点的排列方式不同的现象

类质同象

矿物结构中某种原子或离子部分被他种原子或离子取代，但不破坏其晶体结构的现象。

矿物形态

矿物单体及集合体的外形特征，取决于其内部结构和生成环境。分为自形、他形、半自形

晶体习性

同一种矿物晶体，在一定的外界条件下，趋向于某一种形态的特性
分为一项延长、二项延展、三项等长

矿物的力学性质

解理

矿物受力后沿一定的方向裂开成光滑平面的习性，光滑的平面称为解里面。

断口

外力下解理不发育矿物的不规则裂开

硬度

对外力的抵抗能力

岩石

一种或多种矿物或岩屑组成的集合体，是地质作用的产物，更是地壳岩石圈的组成部分。

  1. 沉积岩：各种外动力地质作用形成的沉淀物在地表或近地表条件下，经过固结成岩作用形成的岩石
  2. 岩浆岩： 熔融状态的岩浆冷凝形成的岩石
  3. 变质岩： 地壳中已形成的岩石在高温、高压及化学活动性流体的作用下，原来的成分、结构、构造等发生变化而形成的岩石。
